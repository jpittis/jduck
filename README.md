# JDuck
A language built in the iconic basement lounge of Queen's Campus.

### Milestone 1 [Complete!]

Given: ```(1 + 10) * (2 + (28 - PI) % 8 * 57) >= 12;```

Produce: ```{>= {* {+ {1} {10}} {+ {2} {% {- {PI}} {* {8} {57}}}}} {12}} ```
